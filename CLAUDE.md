# Glass Pads — Premium Tempered Glass Mousepad Shopify Store

## Project Overview
A modern, luxury, high-converting Shopify store for Glass Pads — premium tempered glass mousepads/desk pads for gamers, creators, and desk setup enthusiasts.

## Brand Positioning
- Premium glass surface with artwork visible underneath
- Smooth consistent glide, easy to clean, long-lasting vs cloth pads
- Modern luxury desk aesthetic — Apple/Razer-inspired minimal
- Target: gamers, creators, professionals, desk setup enthusiasts

## Style Direction
- Modern, clean, futuristic, premium, minimal, high contrast
- Apple/Razer-inspired: not childish, not cluttered
- Strong contrast, smooth sections, premium typography, beautiful product visuals
- Clean spacing, conversion-focused copy, Swiss formalism

## Build Workflow
- Build one page at a time: start with the homepage mockup first
- Every page must be modular — reusable Shopify sections/blocks
- Easy for a non-technical owner to edit via Shopify theme editor
- Avoid fragile code and unnecessary apps
- Responsive for desktop AND mobile
- No Liquid errors, clean section names, clear comments
- Include placeholder image areas for real product photos later

## Technical Standards
- Shopify Liquid + HTML/CSS/JS best practices
- Performance-focused (fast load times, minimal JS)
- Mobile-first responsive design
- Carrier-grade ecommerce conversion patterns
- Section names in Shopify editor must be human-readable

## Memory
- Product: tempered glass mousepad with artwork under glass
- Core selling points: premium surface, smooth glide, easy clean, durable, modern aesthetic
- Customer: gamers (#1), creators, professionals, desk enthusiasts

## Git Workflow
- Branch off main for feature work
- Each major milestone gets its own commit
- User reviews work before merging back to main
